# Executando a Aplicação SmartPurchase-Front com Docker

Este tutorial descreve como construir a imagem Docker da aplicação frontend SmartPurchase-Front e executá-la, tornando-a acessível via navegador na porta 8085.

## Pré-requisitos

*   Docker instalado e em execução em sua máquina.

## Passos para Execução

### 1. Construir a Imagem Docker

Primeiro, você precisa construir a imagem Docker da aplicação. Certifique-se de estar no diretório raiz do projeto (`smartpurchase-front`) onde o `Dockerfile` está localizado.

Execute o seguinte comando no terminal:

```bash
docker build -t smartpurchase-front .
```

Este comando irá:
*   Utilizar o `Dockerfile` para criar uma imagem Docker.
*   Instalar as dependências do Node.js, compilar a aplicação e empacotar os arquivos estáticos.
*   Configurar um servidor Nginx para servir a aplicação.
*   Nomear a imagem resultante como `smartpurchase-front`.

### 2. Executar o Container Docker

Após a construção da imagem, você pode iniciar um container a partir dela. Vamos mapear a porta `8085` da sua máquina local para a porta `80` dentro do container (onde o Nginx está ouvindo).

Execute o seguinte comando:

```bash
docker run -p 8085:80 -d smartpurchase-front
```

Este comando irá:
*   Iniciar um novo container a partir da imagem `smartpurchase-front`.
*   `-p 8085:80`: Mapeia a porta 8085 do seu host para a porta 80 do container.
*   `-d`: Executa o container em modo "detached" (em segundo plano), permitindo que você continue usando o terminal.

### 3. Acessar a Aplicação no Navegador

Com o container em execução, você pode acessar a aplicação SmartPurchase-Front abrindo seu navegador e navegando para:

```
http://localhost:8085
```

Você deverá ver a interface da aplicação SmartPurchase-Front.

## Gerenciamento do Container

Para verificar se o container está em execução:

```bash
docker ps
```

Para parar o container (substitua `<container_id>` pelo ID do seu container, obtido via `docker ps`):

```bash
docker stop <container_id>
```

Para remover o container parado:

```bash
docker rm <container_id>
```

Para remover a imagem Docker (somente se não houver containers usando-a):

```bash
docker rmi smartpurchase-front
```
